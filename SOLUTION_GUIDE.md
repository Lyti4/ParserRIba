# 🔧 Решение проблемы с Camoufox и Пятерочкой

## Проблема
Camoufox не может загрузить Firefox из-за:
1. **Таймаут сети** - GitHub release-assets.githubusercontent.com недоступен или медленно работает из России
2. **Нехватка места** - Требуется ~700 МБ для загрузки Firefox

## ✅ Решение: Используем Playwright вместо Camoufox

Я переключил парсер Пятерочки обратно на **Playwright**, который:
- ✅ Уже установлен и работает
- ✅ Требует меньше ресурсов
- ✅ Не требует дополнительных загрузок

### Изменения в коде:
- `parsers/pyaterochka.py` переключен с `CamoufoxParser` на `BaseParser` (Playwright)
- Метод `fetch_page_camoufox` заменен на `fetch_page`
- Все остальные функции парсинга сохранены

## 🚀 Инструкция для запуска на Windows

```cmd
cd C:\Users\Дима\Desktop\ParserRIba
env\Scripts\activate

# Запуск с видимым браузером (для прохождения капчи)
python main.py --store pyaterochka --no-headless --log-level INFO

# Или в headless режиме (может потребоваться VPN)
python main.py --store pyaterochka --log-level INFO
```

## 💡 Советы для Пятерочки

1. **Первый запуск** - используйте `--no-headless` чтобы увидеть капчу и решить её вручную
2. **После решения капчи** - браузер сохранит куки и следующие запросы будут работать
3. **Если капча повторяется** - попробуйте:
   - Сделать паузу 5-10 минут между запросами
   - Использовать VPN с российским IP
   - Запустить с флагом `--region 77` (Москва)

## 🦊 Когда использовать Camoufox?

Camoufox полезен для сайтов с **очень строгой защитой**:
- Auchan
- Lenta
- Ozon (если потребуется)

Для Пятерочки Playwright работает достаточно хорошо.
