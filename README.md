# Парсер цен на рыбные товары в продуктовых сетях

Приложение для парсинга сайтов продуктовых сетей, сравнения цен и вывода самых низких цен на рыбные товары.

## Возможности

- Парсинг популярных продуктовых сетей Москвы:
  - Пятерочка
  - Магнит
  - Перекресток
  - Лента
  - Ашан (Auchan)
  - Окей (O'KEY)
- Сбор данных о рыбных товарах (название, цена, вес, цена за кг, ссылка, изображение)
- Сравнение цен между магазинами
- Экспорт результатов в Excel таблицу с 4 листами:
  - Все товары
  - Лучшие цены
  - Сравнение по магазинам
  - Статистика
- Асинхронная работа для ускорения парсинга
- Обход базовых защит от парсинга (user-agent, задержки)
- Логирование процесса парсинга

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

1. Создайте файл `.env` на основе `.env.example`
2. При необходимости измените настройки в `config.py`:
   - Список магазинов для парсинга
   - Категории рыбных товаров
   - Таймауты и задержки между запросами

## Использование

```bash
python main.py
```

Результаты сохраняются в файл `fish_prices.xlsx`

## Добавление новых магазинов

Для добавления нового магазина:

1. Создайте новый файл-парсер в директории `parsers/` (например, `newstore.py`)
2. Наследуйтесь от базового класса `BaseParser`
3. Реализуйте методы:
   - `get_category_urls()` - возвращает список URL категорий
   - `parse_fish_products()` - парсит товары из категорий
   - `_extract_product()` - извлекает данные из карточки товара
4. Добавьте импорт и регистрацию парсера в `main.py`
5. Добавьте магазин в список `STORES` в `config.py`

        return self.categories
